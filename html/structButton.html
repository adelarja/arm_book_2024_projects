Detailed Description

Structure to hold buttons data.

It is used to avoid code repetition and to make the code more maintainable.

The struct holds information of the digital pin of the button, the current state of the button (after the debounce filter), the last state of the button (that can or can't be a rebound), the timer used for the debounce function, and some variables to know if the button was pressed and also processed (if the button was pressed we want to process that fact only once, no matter if the button is still pressed).
